## Veröffentlichung über GitHub Pages

1. Erstelle auf github.com ein neues Repository, zum Beispiel `portfolio` oder `gotje-rubarth`.
2. Lade den kompletten Inhalt dieses Ordners hoch, nicht den Ordner selbst:
   - `index.html`
   - Ordner `assets`
   - optional `README.md`
3. Öffne im Repository: `Settings` → `Pages`.
4. Wähle als Source/Branch: `Deploy from a branch`, Branch `main`, Folder `/root`.
5. Speichern.
6. Nach kurzer Wartezeit erscheint dort die öffentliche Website-URL.

## Danach
Die finale URL kann als QR-Code in den CV eingefügt werden.
